# قائمة جاهزية تصدير مزيلات العرق للأسواق الخليجية والعالمية

تأخير يوم في الجمارك يكلّف آلاف الوحدات المبيعة. هذه قائمة جاهزة لتأمين ملفات التصدير قبل شحن مزيلات العرق.

## مستندات أساسية
- **فاتورة تجارية** مع HS Code صحيح (عادة 3307.20).
- **شهادة منشأ** مختومة وغالبًا موثّقة قنصليًا.
- **شهادة تحليل** (COA) لكل دفعة مع تواريخ إنتاج وانتهاء واضحة.

## متطلبات لوائح الخليج
- توافق **SFDA** أو الهيئات المكافئة، مع تسجيل المنتج عند الحاجة.
- ملصق ثنائي اللغة (عربي/إنجليزي) يذكر المكونات، التحذيرات، والكمية الصافية.
- ذكر **درجة الحرارة المثلى للتخزين** لضمان الثبات.

## أسواق أخرى
- **الاتحاد الأوروبي**: ملف PIF، تقييم سلامة، وملائمة الـCPSR.
- **أفريقيا**: قد تتطلب تحاليل محلية؛ احجز مخبرًا مسبقًا لتقليل التأخير.
- **المناطق الحرة**: تأكد من وضوح بيانات إعادة التصدير لمنع الإرجاع.

## لوجستيات وتعبئة
- استخدم **كرتون محكم** يمنع تسرب السبراي أو الرول أون أثناء النقل.
- ضع **ملصقات خطورة الشحن** إن وُجدت مذيبات كحولية بنسبة مرتفعة.
- خطة طوارئ: شحنة صغيرة جوية احتياطية لتغطية النقص إذا تأخر التخليص البحري.

## تحقق قبل الإرسال
1. مراجعة الملصق مع فريق لوائح الاستيراد.
2. تطابق أرقام الدُفعات بين العبوة وCOA.
3. حفظ نسخ إلكترونية لكل مستند في ملف مشترك مع وكيل الشحن.
